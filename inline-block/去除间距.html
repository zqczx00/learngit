display: inline-block 处理去除间距 1、移除前后标签的空格(或者添加注释) 缺点：代码可读性较差 2、使用 margin 负值 .space a { display: inline-block; margin-right: 3px; } 缺点：由于外部环境的不确定性，以及最后一个元素多出的父 margin 值等问题，这个方法不适合大规模使用 3、去除结束标签 111 222 333 (IE6/7需要保留最后一个结束标签) 4、font-size: 0 .space { font-size: 0; -webkit-text-size-adjust: none; (兼容chrome浏览器的默认字体大小) } .space a { font-size: 12px; } (目前 chrome)
